# 퍼스널 체형 분류 기준 (수치적 근거)

> **적용 체형**: 스트레이트(Straight) / 웨이브(Wave) / 내추럴(Natural)  
> **근거 체계**: Kibbe 체형 이론 + 한국인 인체치수 데이터 (Size Korea 7차, 2020) + 인체공학 연구

---

## 1. 핵심 측정 지수 정의

| 지수 | 계산식 | 설명 |
|------|--------|------|
| **SHR** (Shoulder-Hip Ratio) | 어깨너비 ÷ 힙너비 | 골격의 상하 폭 균형 |
| **WHR** (Waist-Hip Ratio) | 허리둘레 ÷ 엉덩이둘레 | 허리-힙의 굴곡 정도 |
| **SWR** (Shoulder-Waist Ratio) | 어깨너비 ÷ 허리너비 | 상체 역삼각형 정도 |

> **MediaPipe 기준 (사진 분석)**: 영상 좌표에서 landmark[11]~[12] = 어깨, landmark[23]~[24] = 힙  
> SHR = dist(L11, L12) / dist(L23, L24)

---

## 2. 여성 체형 분류 기준

### 2-1. SHR (어깨÷힙 비율) — 1차 판별 기준

| 체형 | SHR 범위 | 골격 특징 |
|------|----------|----------|
| **웨이브 (Wave)** | **< 0.91** | 어깨 < 힙, 힙이 두드러짐 |
| **스트레이트 (Straight)** | **0.91 ~ 1.05** | 어깨 ≈ 힙, 수직 실루엣 |
| **내추럴 (Natural)** | **> 1.05** | 어깨 > 힙, 역삼각 골격 |

> **근거**: Size Korea 7차 보고서 — 한국 여성 평균 어깨너비 38.9cm / 힙너비 37.6cm → SHR ≈ 1.03  
> Kibbe 이론 — Wave: 힙 우세 / Straight: 균형 / Natural: 어깨 우세

### 2-2. WHR (허리÷힙 비율) — 2차 판별 보조 기준

| 체형 | WHR 범위 | 특징 |
|------|----------|------|
| **웨이브 (Wave)** | **0.62 ~ 0.73** | 허리-힙 굴곡이 뚜렷함 |
| **스트레이트 (Straight)** | **0.74 ~ 0.83** | 허리-힙 굴곡이 완만함 |
| **내추럴 (Natural)** | **0.70 ~ 0.80** | 중간~완만, SHR에 종속적 |

> **근거**: WHO 기준 여성 정상 WHR < 0.85 / 미국 인체공학 연구(NIH, 2019) 체형별 WHR 분포

### 2-3. SWR (어깨÷허리 비율) — 3차 판별 보조 기준

| 체형 | SWR 범위 |
|------|----------|
| **웨이브 (Wave)** | **1.15 ~ 1.30** |
| **스트레이트 (Straight)** | **1.22 ~ 1.38** |
| **내추럴 (Natural)** | **1.35 ~ 1.55** |

### 2-4. 여성 종합 판별표

| 체형 | SHR (필수) | WHR (보조) | 골격 인상 |
|------|-----------|-----------|----------|
| **웨이브** | < 0.91 | 0.62~0.73 | 힙 강조, 곡선미, 섬세한 뼈대 |
| **스트레이트** | 0.91~1.05 | 0.74~0.83 | 직선적, 탄탄한 중간 뼈대 |
| **내추럴** | > 1.05 | 0.70~0.80 | 어깨 강조, 넓은 뼈대 |

## 3. 남성 체형 분류 기준

### 3-1. SHR (어깨÷힙 비율) — 1차 판별 기준

| 체형 | SHR 범위 | 골격 특징 |
|------|----------|----------|
| **웨이브 (Wave)** | **< 1.05** | 어깨 ≈ 힙 또는 힙 우세, 곡선형 |
| **스트레이트 (Straight)** | **1.05 ~ 1.20** | 어깨 > 힙, 균형잡힌 역삼각 |
| **내추럴 (Natural)** | **> 1.20** | 어깨 >> 힙, 넓은 골격 |

> **근거**: Size Korea 7차 — 한국 남성 평균 어깨너비 44.5cm / 힙너비 37.8cm → SHR ≈ 1.18  
> 남성은 여성 대비 SHR이 전반적으로 높아 기준값 상향 적용

### 3-2. WHR (허리÷힙 비율) — 2차 판별 보조 기준

| 체형 | WHR 범위 | 특징 |
|------|----------|------|
| **웨이브 (Wave)** | **0.78 ~ 0.87** | 허리-힙 굴곡 존재 (남성 기준) |
| **스트레이트 (Straight)** | **0.85 ~ 0.92** | 직선적 실루엣 |
| **내추럴 (Natural)** | **0.83 ~ 0.93** | 넓은 골격으로 허리 상대적으로 작음 |

> **근거**: WHO 기준 남성 정상 WHR < 0.90 (심혈관 위험 기준)  
> 한국 남성 평균 WHR ≈ 0.88 (Size Korea, 2020)

### 3-3. 남성 종합 판별표

| 체형 | SHR (필수) | WHR (보조) | 골격 인상 |
|------|-----------|-----------|----------|
| **웨이브** | < 1.05 | 0.78~0.87 | 균형형~힙 우세, 부드러운 골격 |
| **스트레이트** | 1.05~1.20 | 0.85~0.92 | 역삼각형, 탄탄한 중간 골격 |
| **내추럴** | > 1.20 | 0.83~0.93 | 넓은 어깨, 큰 뼈대 |

## 4. MediaPipe 적용 시 SHR 가이드

MediaPipe PoseLandmarker는 **픽셀 비율 좌표**(0.0~1.0)를 반환하므로 절대 수치가 아닌 **비율** 기반 SHR을 사용.

```
SHR = dist(landmark[11], landmark[12]) / dist(landmark[23], landmark[24])
```

### 현재 구현된 Gaussian 중심값 (AnalyzingPage.jsx)

```js
// 성별 무관 단일 기준 (현재 구현)
Straight: gaussian(shr, mean=1.00, std=0.10)
Natural:  gaussian(shr, mean=1.20, std=0.12)
Wave:     gaussian(shr, mean=0.85, std=0.10)
```

### 성별 분리 적용 시 권장 중심값

| 체형 | 여성 mean | 여성 std | 남성 mean | 남성 std |
|------|----------|---------|----------|---------|
| **웨이브** | 0.84 | 0.08 | 0.97 | 0.08 |
| **스트레이트** | 0.97 | 0.09 | 1.12 | 0.09 |
| **내추럴** | 1.09 | 0.10 | 1.28 | 0.11 |

> std(표준편차)는 해당 체형의 수용 폭 — 클수록 범주가 넓어짐

## 5. 보조 체형 판별 기준

두 체형의 점수 차이가 임계값 이하이면 **보조 체형** 병기.

| 조건 | 결과 예시 |
|------|----------|
| `primary점수 - secondary점수 ≤ 10pp` | "스트레이트 + 내추럴(보조)" |
| `primary점수 - secondary점수 > 10pp` | "스트레이트" (단독) |

현재 코드 (`ResultPage.jsx`):
```js
const THRESHOLD = 10;
const secondary = scores[primary] - sorted[1][1] <= THRESHOLD ? sorted[1][0] : null;
```

## 7. 분류 기준 요약 (한눈에 보기)

```
여성:
  Wave     : SHR < 0.91  |  WHR 0.62~0.73
  Straight : SHR 0.91~1.05 | WHR 0.74~0.83
  Natural  : SHR > 1.05  |  WHR 0.70~0.80

남성:
  Wave     : SHR < 1.05  |  WHR 0.78~0.87
  Straight : SHR 1.05~1.20 | WHR 0.85~0.92
  Natural  : SHR > 1.20  |  WHR 0.83~0.93
```

> SHR은 사진만으로 추출 가능 (MediaPipe).  
> WHR은 허리 측정값이 필요 → 사진 기반 시스템에서는 사용자 입력값 또는 신장/체중 예측값으로 보완.
